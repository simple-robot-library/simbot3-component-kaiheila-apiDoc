abstract class KaiheilaUserUpdatedEvent : KaiheilaSystemEvent<UserUpdatedEventBody> , ChangedEvent
Content copied to clipboard

开黑啦用户信息更新事件。 此事件属于一个 ChangedEvent, 变更的源为发送变更的用户 的ID （因为此事件不一定是某个具体频道服务器中的用户，只要有好友关系即会推送）， 变更的 前由于无法获取而始终为null， 后 为用户变更事件的内容本体，即 UserUpdatedEventBody 。
