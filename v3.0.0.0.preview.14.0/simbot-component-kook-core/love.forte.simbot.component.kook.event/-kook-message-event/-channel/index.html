频道消息事件。

此类型可能是 KookChannelMessageEvent, 则代表为一个普通的频道成员发送的消息事件； 或者是 KookBotSelfChannelMessageEvent, 则代表为bot自己所发出的消息。

普通成员消息 会实现 ChannelMessageEvent, 但是 bot频道消息 只会实现基础的 MessageEvent、ChannelEvent、MemberEvent.
