open override fun getContact(id: ID): KaiheilaUserChat
Content copied to clipboard

通过指定ID 构建 一个目标用户的聊天会话对象。

聊天会话 目前不会进行缓存，每次获取都会通过 UserChatCreateRequest 请求并构建新的实例。 由于每次都会通过api请求，因此不存在 "没找到" 的情况，getContact 将不会返回非null值。

但是会抛出任何可能由 UserChatCreateRequest 抛出的或者请求过程中产生的任何异常。
