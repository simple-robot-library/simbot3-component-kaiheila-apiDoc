开黑啦的 user-chat 私聊会话。

开黑啦组件会将私聊会话视为 Friend 处理

开黑啦组件会将 私聊会话 视为 Stranger 处理，但是会同时实现 Contact 来提供可交流的联系人能力。

可删除的

开黑啦中的聊天会话是可以通过 UserChatDeleteRequest 进行删除的。因此 KaiheilaUserChat 实现了 DeleteSupport 来支持 删除操作。
